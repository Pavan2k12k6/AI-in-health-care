Import Patients

Upload Patient Data CSV

Import multiple patients and their associated data from a CSV file. This will automatically create patient records in the system.

The CSV file should have the following required columns:

  • patient_id - Unique identifier for the patient
  • first_name - Patient's first name
  • last_name - Patient's last name
  • date_of_birth - Patient's date of birth (YYYY-MM-DD format)
  • gender - Patient's gender

Optional columns include:

  • weight - Patient's weight in kg
  • medical_conditions - List of medical conditions (comma-separated or JSON format)
  • genotype_data - Genotype data in format: gene:variant,gene:variant or as JSON
Only CSV files are accepted.
CSV Format Example

Here's an example of how your CSV file should be structured:

patient_id,first_name,last_name,date_of_birth,gender,weight,medical_conditions,genotype_data
P12345,John,Doe,1980-01-15,Male,75.5,"Hypertension,Diabetes","CYP2D6:*1/*4,CYP2C19:*1/*2"
P12346,Jane,Smith,1975-05-20,Female,65,"Asthma,Arthritis","CYP2D6:*1/*1,CYP2C19:*1/*1"

For more complex data, you can use JSON format in the medical_conditions and genotype_data columns:

patient_id,first_name,last_name,date_of_birth,gender,weight,medical_conditions,genotype_data
P12347,Alice,Johnson,1990-10-10,Female,62,"[{\"name\":\"Migraine\",\"diagnosis_date\":\"2019-03-15\",\"notes\":\"Chronic\"}]","[{\"gene\":\"CYP2D6\",\"variant\":\"*1/*4\",\"phenotype\":\"Intermediate Metabolizer\"}]"
